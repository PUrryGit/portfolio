Personal Projects

Command.io

Command.io is a Discord bot that allows the user to play local music files to a server over voice chat. It has simple playlist functionality, such as queueing up songs and playing random songs amongst the queue.
This Discord bot came out of a need for sharing thematic music during sessions of Dungeons & Dragons. After YouTube and Spotify removed their free legacy APIs, many music playing bots were taken offline. This led me to learning DiscordJS and creating a bot that played local files, so I wouldn't have to rely on file API access. The bot was originally created in DiscordJS v13, and upgraded to DiscordJS v14 when the API v13 used was sunset. For a full list of features, you can find the project on the GitHub page.
Examples of some features, as shown in the image on the left:

/roll - Command.io has the ability to simulate dice rolling for custom-sized dice. For those particular cases where you may need a 1d2, or perhaps even a 7d7. Limits at d1000 for message overflow.

/rd20, /rd12, /rd10 etc. - The bot comes pre-bundled with a number of /roll commands for popular dice sizes.

/play - When given a filepath to a music file, Command.io will load the file into an AudioPlayer to play over a Discord channel. When used while another song is played, it will queue the songs into a playlist.

/next - Moves the needle to the next file in the playlist.
